Pyramid Day 1

Monday, October 30

Today was the first day of learning how to use the Pyramid framework. The lecture moved pretty fast but I still appreciate that we went through the code in the demo. Honestly, right now Pyramid has a lot of moving parts that I don't fully understand to be honest, but I think I have a general understanding that I hope to build on in the coming days. The idea that each route has an associated function or some sort of handler makes sense, but actually keeping track of all the files and objects is the part that's the most confusing. Hopefully it gets better with repeated exposure.
